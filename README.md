# Countdown Timer

A countdown timer project built using HTML, CSS and JavaScript.

## Features
- Shows remaining days, hours, minutes and seconds
- Real-time countdown update
- Responsive design
- Uses JavaScript Date and Time functions

## Technologies Used
- HTML
- CSS
- JavaScript
